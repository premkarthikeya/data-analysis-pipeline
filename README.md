# Data Analysis Pipeline Project

## Overview
This project demonstrates an end-to-end data analysis pipeline using Python, SQL, and Power BI.

## Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI

## Steps Performed
- Data cleaning and preprocessing
- SQL querying (joins, aggregations)
- Data analysis and insights generation
- Visualization using Power BI

## Outcome
Extracted meaningful insights such as trends, comparisons, and performance metrics.
